# Farming Improvements

The Improving Minecraft mod adds several changes to farming mechanics.

## Bonemeal Changes

| Feature | Status |
|---------|--------|
| One bonemeal per sapling | ✅ Enabled |
| Bonemeal on sugar cane | ✅ Enabled |
| Bonemeal on cactus | ❌ Disabled |
| Bonemeal clones lily pads | ❌ Disabled |

> [!TIP]
> A single bonemeal is usually enough to fully grow a tree from a sapling.

## Nether Wart Fertilizers

Gunpowder, Magma Cream, and Blaze Powder can be used as fertilizers for Nether Wart - the same way bonemeal works on other plants.

## Feathers

Chickens leave feathers on the ground from time to time (configurable frequency).

## Stackable Items

The following items are stackable to 16:
- Cakes
- Boats
- Minecarts
- Beds

## Additional Options (disabled by default)

These features exist in the config but are **disabled** in NTNH:

| Feature | Effect |
|---------|--------|
| All crops need one bonemeal | Overrides normal growth mechanics |
| Dangerous fire | Faster fire spread, can't punch out |
| Logs need axes | Can't punch wood bare-handed |
| No wooden tools | Removes wooden tool recipes |
| Hard recipes | Tougher nametag/saddle/horse armor recipes |
