# Resource Pack Update Notifier
The <Color id="GREEN">Resource Pack Update Checker</Color> scans your enabled resource packs and checks GitHub releases for newer compatible versions. If an update is found, you receive an in-game chat message with a clickable link to the newest release. This requires opt-in from the resource pack creator.
