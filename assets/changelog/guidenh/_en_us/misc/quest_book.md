# Quest Book Features
The quest book includes arrows on lines connecting quests to help visualize the flow of dependencies. Hovering over a quest animates all connected lines. A button on the left sidebar toggles the arrows on and off.

> [!NOTE]
> The animation can be disabled in the config if you so desire
