# InGame Info Toggling
<Color id="GREEN">InGame Info</Color> (the information in the top left by default) supports a second config file, allowing you to keep two different HUD layouts and swap between them at any time in-game.

Add an optional second file (`\config\InGameInfo2.xml`) and use the toggle key to cycle: primary -> secondary -> OFF!
