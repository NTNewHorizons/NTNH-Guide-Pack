# Angelica Changes
<Color id="GREEN">Angelica</Color> is a rendering overhaul mod that replaces Optifine, improving rendering across the board and providing built-in shader compatibility.

## Compact CTM
For texture developers, Angelica provides built-in support for <Color id="GREEN">Compact CTM</Color>. Instead of needing 47 unique textures per block, you only need 5 files - a ninefold reduction in workload and file size with no loss in visual quality.

## Advanced Color Tracing
> [!WARNING]
> This feature is still in development.
Work is ongoing to bring <Color id="GREEN">Advanced Color Tracing</Color> as a default rendering feature. Those using shader packs like Euphoria Patches will already be familiar with colored lighting effects.
